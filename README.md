# Loadout-Swapper

Modify your active loadout at any time,. This mod allows dynamic on-the-fly adjustments without returning to menus, giving full control over your build during gameplay.

## Features
- Swap melee, primary and secondary weapons instantly.
- Swap throwables, armor and equipment.
- Swap ammo type.

## Download

You can download the latest version of the trainer [here](https://github.com/AntonisKazantzis/Loadout-Swapper/releases/download/v1.0/Loadout.Swapper.zip).

### Installation Guide

1. Download the zip file.
2. Extract the contents and copy them into your Payday 2 mods directory.
3. Launch Payday 2, navigate to the mods menu, and assign a keybind to activate the Trainer.
